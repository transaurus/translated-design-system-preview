## 概要

アイテムのリストを表示するページです。各アイテムは詳細ビューまたは作成ビューを表示できる新しいページへのリンクとなっています。

    <ListPage />
  </DataProvider>
</Page>

## 使用方法

- `Stack`コンポーネントと`column`方向、`Card`コンポーネントを組み合わせてリストレイアウトを作成します。
- `Card`コンポーネントを使用してリストアイテムを作成します。
- 空の状態を表示するには、`EmptySearchResult`コンポーネントを使用します。
- フィルターが存在する場合、主要なアクションはフィルターの後、ページの右側に配置する必要があります。
