# リソース <Badge text='ready' color='green'></Badge>

Sagaデザインシステムを使用してより良いソリューションを構築するために、デザイナーや開発者向けにいくつかのツールを用意しています。

## 内部デザイナー向けリソース

これらのFigmaライブラリを使用して、Grafanaのエクスペリエンスをデザインおよびモックアップできます。

_現時点ではこれらは**内部限定**のリソースです。_
