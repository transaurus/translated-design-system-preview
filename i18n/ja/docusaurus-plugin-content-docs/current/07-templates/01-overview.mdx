# Saga デザインシステム テンプレート

Sagaデザインシステムのテンプレートは、Grafanaエコシステム内でユーザーインターフェースを構築するための堅牢な出発点を開発者やデザイナーに提供することを目的とした、事前に構築されたパターンとコードフレームワークのセットです。これらのテンプレートは、デザインシステム内の個々のコンポーネントよりも高レベルの抽象化を提供し、UI開発をより効率的に進めるためのアプローチを可能にします。

Sagaデザインシステムのテンプレートは、以下の課題に対処することを目的としています：

- 信頼性のあるフレームワークと標準化されたパターンの欠如により、リストや表形式データを含むページなど類似したページタイプ間でUIの一貫性が失われ、さまざまなページで同様の機能が異なる見た目で実装される。これにより、断片化されたユーザー体験が生まれるだけでなく、ユーザーの認知的負荷が増大するという問題が発生する。
- デザイナーや開発者は、以前に実装されたことのある一般的なレイアウトを含む新しいUI機能を実装する際に、多大な労力を費やす必要がある。このような余分な重複作業は、UI開発プロセスを必要以上に複雑にしてしまう。

## テンプレートを使用するタイミング

以下の場合にテンプレートを使用してください：

- 既存のページや機能と類似した新しいページや機能を作成する場合
- 以前に実装されたことのある一般的なレイアウトやパターンを実装する場合
