# ボイスとトーン <Badge text='ready' color='green'></Badge>

Grafanaでは、会社のコミュニケーションをあたかも実際の人物が話しているように表現することを好みます。

ボイスとトーンは、あなたの態度と情報の伝え方を反映します。単語の選択、太字や斜体の使用、句読点、文法はすべてボイスに影響を与えます。選択するトーンはメッセージの緊急性を反映し、状況に応じて変化する可能性があります。

技術文書を作成する際は、[ボイスとトーンのガイドライン](https://grafana.com/docs/writers-toolkit/write/style-guide/voice-tone-guidelines/)を参照してください。
