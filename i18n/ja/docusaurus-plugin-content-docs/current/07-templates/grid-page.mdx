## 概要

グリッドレイアウトでカードアイテムのリストを表示するページ。各カードは、詳細ビューまたは作成ビューを表示できる新しいページへのリンクです。

## 使用方法

- `Grid`コンポーネントと`Card`コンポーネントを組み合わせて、カードのグリッドレイアウトを作成します。
- `Card`コンポーネントを使用してグリッドアイテムを作成します。
- レスポンシブなレイアウトにするには、`Grid`コンポーネントの`minColumnWidth`プロパティを使用します。
- 固定数の列が必要な場合は、`Grid`コンポーネントの`columns`プロパティを使用します。
