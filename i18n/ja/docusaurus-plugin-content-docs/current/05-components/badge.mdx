# バッジ <Badge text='draft' color='orange'></Badge> <a href='https://developers.grafana.com/ui/latest/index.html?path=/story/data-display-badge--basic' target='_blank' className='header-links'>Storybook <Icon name="external-link-alt"/> </a>

バッジコンポーネントは、リリースステータスや新規要素など、他のコンテンツに関するメタ情報を追加します。任意の `Icon` コンポーネントを追加したり、アイコンなしでバッジを使用することも可能です。

## オプション

### カラー

バッジは、外観を変更するために名前付きのビジュアライゼーションカラーを受け入れます。これらのカラーは、successやerrorなどのデザインシステムトークンとは異なることに注意してください。

```jsx live
<Stack direction="horizontal">
  <Badge color="blue" text="Blue" />
  <Badge color="red" text="Red" />
  <Badge color="green" text="Green" />
  <Badge color="orange" text="Orange" />
  <Badge color="purple" text="Purple" />
</Stack>
```

### アイコン

バッジの先頭に表示するアイコン名を指定できます。

```jsx live
<Badge icon="rocket" color="blue" text="New!" />
```

### ツールチップ

ホバー時に追加情報を表示するツールチップを指定します。ツールチップに長いコンテンツを配置しないようにしてください。

```jsx live
<Badge icon="rocket" color="blue" text="New!" tooltip="Check out this new feature!" />
```
