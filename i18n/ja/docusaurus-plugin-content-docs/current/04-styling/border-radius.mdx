# ボーダー半径 <Badge text='draft' color='orange'></Badge>

角丸は主にスタイリッシュな選択肢であり、Grafana UIをよりモダンに見せ、親しみやすさを感じさせ、視覚要素に「洗練された」感覚を加えます。

角丸の計測はボーダー半径と呼ばれ、ピクセルまたはパーセンテージで定義できます。

### デフォルトのボーダー半径

これは一般的なボーダー半径または`shape.radius.default`で、2pxと定義されており、ボタン、ポップオーバーパネル、ツールチップ、ウィジェット、ダイアログなど、ほとんどのUI要素に使用されます。

![標準的なボーダー半径](/img/shape.radius.default.png 'Grafana UIにおける標準的なボーダー半径の視覚的描写')

### ピル型ボーダー半径

ピル型ボーダー半径または`shape.radius.pill`は、9999pxと定義されています。このスタイリングはGrafana UIではあまり見られません。スイッチなどの小さなUI要素が、丸みを帯びた要素の周りに滑らかでクリーンなエッジを持つためにピル半径を使用します。

![Do](/img/shape.radius.pill_do.png)
Smaller elements use the `pill` styling

![Dont](/img/shape.radius.pill_dont.png)

### 円形ボーダー半径

円形ボーダー半径または`shape.radius.circle`は、100%と定義されています。このスタイリングもGrafana UIではあまり見られず、要素に完全な円形を提供します。

![円形ボーダー半径](/img/shape.radius.circle.png 'Grafana UIにおける円形ボーダー半径の視覚的描写')
