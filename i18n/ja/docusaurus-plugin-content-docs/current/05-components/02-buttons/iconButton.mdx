# IconButton <Badge text='ready' color='green'></Badge> <StorybookLink path='/story/buttons-iconbutton--basic'/>

`IconButton`はアイコンのみを表示する操作可能なボタンです。標準的なラベルがないため、ホバー時にラベルを表示できるよう、すべてのIconButtonにツールチップを含めることが重要です。

```tsx live
<IconButton tooltip="Add new" name="plus" />
```

### 推奨事項

- 必ずボタンのラベルをツールチップに含めてください。
- ツールチップが意味をなさず、アイコンが明確な場合（例：フォルダを展開するための<Icon name="angle-down" />など）は、aria-labelを追加し、ツールチップを削除します。
- `IconButton`をテキストと一緒に使用する場合、`display: flex`と`align-items: center`を設定したラッパーを追加して、整列させるようにしてください。
- スペースが限られており、コンテキストが少ない環境で使用してください。
- アイコンの意味が明確で曖昧でないことを確認してください。
- アイコンのみが必要な場合は、`Button`の代わりに`IconButton`を使用してください。

### 非推奨事項

- コンテキストが複雑な状況での使用は避けてください。

## タイプと動作

### バリアント

| Type            | Style | Purpose                                                                                                                                           |
| --------------- | ----- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Primary**     | Ghost | Used for primary-based actions. Only use one per section. If there is no primary action, all Button components should be secondary.               |
| **Secondary**   | Ghost | This is the default-styling for buttons. It's used most frequently within Grafana and can trigger a variety of actions.                           |
| **Destructive** | Ghost | This is used for delete actions.                                                                                                                  |
| **Disabled**    | Ghost | Use this if the functionality the IconButton provides is not currently available, but the IconButton needs to indicate that this function exists. |

### 構成要素

- アイコン
- コンテナ（ホバー時）

### 配置

`IconButton`は、スペースが限られており、操作が多い領域で最も効果的に使用されます。これにはツールバーや、テーブルやカードなどのインライン編集を提供する領域が含まれます。

### ボタンサイズ

| Icon size | Name & size in Figma | Name & size in code |
| --------- | -------------------- | ------------------- |
| 12px      | **small:** 20px      | **xs:** 20px        |
| 14px      |                      | **sm:** 22px        |
| 16px      | **medium:** 28px     | **md:** 24px        |
| 18px      |                      | **lg:** 26px        |
| 24px      | **large:** 48px      | **xl:** 32px        |
| 24px      |                      | **xxl:** 32px       |
| 24px      |                      | **xxxl:** 32px      |

### アクセシビリティ

- ツールチップ用のテキストを追加してください（これはaria-labelとしても使用されます）
- ツールチップが意味をなさない例外的な場合は、aria-labelを追加してください。

## 動作

### ホバー

ホバー時、`IconButton`はIconButtonのラベルを含むツールチップを表示します。
